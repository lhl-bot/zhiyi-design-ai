## 新增三大功能模块

### 1. 左侧边栏导航系统
- 深色侧边栏（72px 宽，hover 展开至 160px），固定在左侧
- 四个视图入口：工作台、出款看板、客户对比、灵感板
- 使用 lucide-react 图标（Sparkles, LayoutDashboard, Users, Palette）
- 移动端自动变成底部固定导航栏

### 2. 多客户出款看板 (DashboardView)
- 全局 KPI：总出款数、已选精修池数、平均分、覆盖客户数
- 每位客户一张卡片，显示：名称/市场、选入进度条、统计数据、最近款式缩略图
- 市场筛选 + 点击客户卡片直接跳转到工作台并选中该客户
- 文件：src/components/DashboardView.tsx

### 3. 客户对比视图 (ComparisonView)
- 支持同时对比 2-3 位客户
- 四个 Tab：总览（侧表对比 + 共性/差异分析）、风格标签、价格策略、趋势判断
- 自动计算客户之间的共同标签和面料差异
- 文件：src/components/ComparisonView.tsx

### 4. 灵感收集板 (MoodBoardView)
- 多面板管理（类似画布标签页）
- 支持从出款图库选择添加、本地上传参考图
- 画布上自由拖拽排列（鼠标拖拽 move）
- 每个图片可编辑标注，标注来源（AI生成 / 本地上传）
- 数据持久化到 localStorage
- 文件：src/components/MoodBoardView.tsx

### 技术细节
- App.tsx 新增 `ViewType` 联合类型和 `activeView` 状态
- 使用条件渲染切换视图，保持原有工作台内容不变
- 新增 CSS 约 500 行（侧边栏/看板/对比/灵感板全部样式）
- 移动端响应式完整适配
- TypeScript 编译通过，Vite dev server 运行正常
