- `DELETE /api/img/:file` — 同时删除关联的 `.json` 元数据

### 前端改动

- `src/App.tsx` — 启动时自动从后端恢复已保存图片为 looks（按 customerId 关联）；生成时传递客户信息
- `src/components/ImageGalleryView.tsx` — 新建「图片库」页面，按客户分组展示所有已保存图片，支持大图预览/下载/删除

- `src/styles.css` — 新增 ~200 行图片库样式

## 修复：线稿图语言修改（SketchChat）不能用

**问题：** 线稿图对话修改功能调用火山方舟 API 时，`referenceImage` 传入的是本地 `/api/img/xxx.jpeg` 或 SVG data URI，火山方舟服务器无法访问 localhost，且不支持 SVG 作为图生图输入。

### 修复（`server/erp-readonly.mjs` POST /api/genimage）
- `/api/img/xxx.jpeg` → 从磁盘读取文件，转为 base64 data URI 后传给火山方舟
- `data:image/svg+xml` → SVG 不被 Ark API 接受，跳过参考图，仅用 prompt
- `data:image/...;base64,...` → 直接透传
- 公网 URL → 先尝试下载转 base64，失败则用原始 URL
- 转换失败时降级为不带参考图生成

### 测试
- npx tsc --noEmit 零错误
- vite build 通过
